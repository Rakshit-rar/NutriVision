# NutriVision AI 🍎🤖

NutriVision AI is a production-ready multimodal food recognition and nutrition intelligence platform that combines state-of-the-art computer vision, vision-language models, and nutritional analytics to help users make informed dietary decisions.

Users can upload a food image, and the system leverages Florence-2, CLIP, Qwen2-VL, and LLaVA to identify the food, understand its ingredients, estimate serving size, and retrieve detailed nutritional information. The platform automatically logs meals, tracks daily nutrient intake, generates interactive analytics dashboards, and provides AI-powered health insights and recommendations.

### Key Features

* 📸 AI-powered food recognition from images
* 🧠 Multimodal food understanding using Vision-Language Models
* 🥗 Ingredient detection and serving size estimation
* 🔥 Nutrition analysis (Calories, Protein, Carbs, Fat, Fiber, Sugar, Sodium)
* 📊 Interactive nutrition analytics dashboard
* 🗓️ Daily, weekly, and monthly nutrition tracking
* 💾 SQLite-based meal logging system
* 🇮🇳 Extensive support for Indian and global cuisines
* 💡 Personalized AI health insights and recommendations

### Tech Stack

Python • Streamlit • SQLite • Pandas • Plotly • OpenCV • Florence-2 • CLIP • Qwen2-VL • LLaVA • Hugging Face Transformers • USDA FoodData Central API

NutriVision AI transforms food images into actionable nutritional intelligence, enabling smarter eating habits through the power of multimodal AI.
